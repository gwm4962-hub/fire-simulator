# High-Precision Asset Formation Simulator (FIRE Simulator)

This is an advanced asset lifespan simulation tool using Markov transition models and Monte Carlo methods.

## 🚀 Features
- **Regime Transition Model**: Reproduces economic fluctuations (bullish and bearish markets) using Markov chains to predict asset trends more realistically.
- **Monte Carlo Method**: Statistically calculates the probability of success through thousands of trials.
- **Survival Probability Model**: Considers age-dependent mortality risk based on the Gompertz-Makeham distribution.
- **Sensitivity Analysis Heatmap**: Visualizes the impact of changes in stock ratios during working years and after retirement on the success rate.

## 🛠 Technologies Used
- HTML5 / CSS3 (Modern UI)
- JavaScript (Vanilla JS)
- Chart.js (Visualization of asset trends)
- Canvas API (Drawing of sensitivity analysis heatmap)

## 📈 Built-in Mathematical Models
1. **MPT (Modern Portfolio Theory)**: The concept of an efficient frontier based on mean and variance.
2. **Markov Transition**: Controlling economic cycle transitions with a probability matrix.
3. **Gompertz-Makeham Distribution**: Precise life expectancy setting based on demographics.

# 高精度資産形成シミュレーター (FIRE Simulator)

マルコフ遷移モデルとモンテカルロ法を用いた、高度な資産寿命シミュレーションツールです。

## 🚀 特徴
- **レジーム遷移モデル**: 景気変動（強気相場・弱気相場）をマルコフ連鎖で再現し、より現実に近い資産推移を予測。
- **モンテカルロ法**: 数千回の試行により、成功確率を統計的に算出。
- **生存確率モデル**: Gompertz-Makeham分布に基づき、年齢に応じた死亡リスクを考慮。
- **感度分析ヒートマップ**: 現役時とリタイア後の株式比率の変化が成功率に与える影響を可視化。

## 🛠 使用技術
- HTML5 / CSS3 (Modern UI)
- JavaScript (Vanilla JS)
- Chart.js (資産推移の可視化)
- Canvas API (感度分析ヒートマップの描画)

## 📈 組み込まれた数理モデル
1. **MPT (現代ポートフォリオ理論)**: 平均・分散に基づいた効率的フロンティアの概念。
2. **マルコフ遷移**: 景気の切り替わりを確率行列で制御。
3. **Gompertz-Makeham分布**: 人口統計に基づいた精密な寿命設定。
